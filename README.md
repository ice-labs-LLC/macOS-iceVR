# macOS-iceVR
A macOS VR API for Meta Quest and Pico.
# IMPORTANT INFO: SUPPORTED ENGINES
We will only support Unity and Xcode (with correct frameworks)
# IMPORTANT INFO: SUPPORTED MACS
From the start, only Apple Silicon will be supported. And Intel support is most likely unlikely as Apple is deprecating Intel support. And if we do release an Intel version, you will probably need a dedicated GPU Intel Mac, some features will probably be disabled.
